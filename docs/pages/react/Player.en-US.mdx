```ts
import { Player } from '@livepeer/react';
```

## Usage

```tsx
import { Player } from '@livepeer/react';

const playbackId =
  'bafybeida3w2w7fch2fy6rfvfttqamlcyxgd3ddbf4u25n7fxzvyvcaegxy';

function App() {
  return <Player playbackId={playbackId} />;
}
```

## Configuration

### playbackId or src

A playback ID for an [Asset](https://github.com/livepeer/livepeer.js/blob/main/packages/core/src/types/provider.ts) or
[Stream](https://github.com/livepeer/livepeer.js/blob/main/packages/core/src/types/provider.ts), or a source URL. One of these is
required. If a playback ID is provided, livepeer.js will automatically poll the provider for a
playback URL corresponding to the playback ID.

```tsx {6}
import { Player } from '@livepeer/react';

const playbackId = 'abcde6mykgkvtxav';

function App() {
  return <Player playbackId={playbackId} />;
}
```
