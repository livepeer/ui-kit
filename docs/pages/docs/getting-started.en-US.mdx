# Getting Started

<Callout>
  These docs are written for a React developer building on livepeer.js. For Node.js or alternative frameworks, you can use the
  vanilla JS library.
</Callout>

## Installation

Install `livepeer` and `@livepeer/react`, and their `wagmi` and `ethers` peer dependencies.

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab>
```bash
npm i livepeer @livepeer/react wagmi ethers
```
  </Tab>
  <Tab>
```bash
pnpm i livepeer @livepeer/react wagmi ethers
```
  </Tab>
  <Tab>
```bash
yarn add livepeer @livepeer/react wagmi ethers
```
  </Tab>
</Tabs>

## Quick Start

### 1. Create a livepeer client

First, create a livepeer `Client` instance using [`createReactClient`](/docs/client), and pass a provider to it.

```tsx
import { createReactClient } from '@livepeer/react';
import { studioProvider } from 'livepeer/providers/studio';

const client = createReactClient({
  provider: studioProvider({ apiKey: 'yourStudioApiKey' })
});
```

Note: If you choose to use Studio as your provider, you will need to configure an API key for the
`studioProvider` which is [CORS protected](https://docs.livepeer.studio/guides/livestreaming/api-key#cors).

[Read more about client configuration](/docs/client)

### 2. Wrap app with `LivepeerConfig`

Next, wrap your app with the [`LivepeerConfig`](/docs/LivepeerConfig) component, passing the `client` to it.

```tsx
import { LivepeerConfig } from '@livepeer/react';

function App() {
  return (
    <LivepeerConfig client={client}>
      <SomeComponents />
    </LivepeerConfig>
  )
}
```

### 3. Enjoy!

Use hooks! Every component inside the `LivepeerConfig` is now set up to use the livepeer hooks.

```tsx
import { useLPMSProvider } from '@livepeer/react';

function LivepeerMediaServer() {
  const lpmsProvider = useLPMSProvider();

  return <div>Using {lpmsProvider.getLPMS().name}</div>
}
```

Want to learn more? Continue reading the documentation for more details.
